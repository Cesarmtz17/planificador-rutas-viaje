# Planificador de Rutas de Viaje

Sistema que optimiza rutas turísticas usando algoritmos como **Vecino Más Cercano**. Lee datos de Excel y calcula la mejor ruta considerando distancias y tiempos de visita.

## Tecnologías
- Java 11+
- Apache POI (Excel)
- Maven

## Ejecutar
```bash
mvn compile exec:java
```
